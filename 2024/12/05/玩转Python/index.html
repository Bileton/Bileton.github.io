配置环境变量

需要配置的环境变量是python的安装路径和它的Scripts目录。

python的安装路径下有python.exe。
Scripts目录下有pip.exe和我们安装的第三方库。

HTTP服务器

命令行运行python -m http.server [port]，创建http服务器，默认端口是8000。

使用virtualenv来创建虚拟环境。

创建虚拟环境

会在当前目录下创建一个文件夹，里面包含了Python解释器的副本和一套标准库。

激活虚拟机环境

在虚拟环境目录下的Scripts目录里，有一个activate.exe，运行它，虚拟环境就生效了。

激活虚拟环境后，你会看到命令行提示符前出现环境名称，表明当前是在该虚拟环境中操作。
