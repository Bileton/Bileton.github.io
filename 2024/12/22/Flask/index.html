路由

route()

使用route()装饰器来把函数绑定到 URL

路由参数

<variable_name>路由参数

通过<>标记的变量作为参数传递给函数。

converter:variable_name参数类型

通过转换器converter为参数指定类型。默认类型为string即字符串。

转换器类型

converter 描述
string 文本
int 整数
float 浮点数
path 路径
uuid UUID字符串

请求方式

通过定义路由来指定请求方式，Flask默认将路由处理为GET请求。
