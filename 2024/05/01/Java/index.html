Byte

byte 类型占 1 个字节（8 位），取值范围从-128（0b10000000）到 127（0b01111111），它在内存中以补码的形式存储整数。

lastIndexOf()

返回指定字符（Unicode代码点）在字符串中最后一次出现的索引位置。

equalsIgnoreCase()

面向对象

Interface（接口）

定义接口

接口的定义使用interface关键字。接口中的方法默认是public和abstract，不需要显式声明。

接口的实现

通过implements关键字实现接口。实现接口的类必须提供接口中所有方法的具体实现。

接口的继承

接口也可以继承其他接口。一个接口可以扩展多个接口，这意味着一个类可以实现多个接口，从而实现多继承。

InnerClass（内部类）

成员内部类

成员内部类是定义在外部类内部、方法外部的类。

成员内部类可以直接访问外部类的所有成员（包括私有成员）。

必须通过外部类对象来创建内部类对象。

内部类会隐式持有一个外部类对象的引用。

静态内部类

使用 static 修饰的内部类称为静态内部类。

静态内部类不依赖于外部类的实例，因此无法访问外部类的非静态成员。

不需要外部类的实例即可创建静态内部类对象。

只能访问外部类的静态成员。

局部内部类

局部内部类是在外部类的方法或作用域内定义的类。

它的作用范围仅限于定义它的方法或代码块中。

局部内部类可以访问所在方法的局部变量，但这些变量必须是 final 或 effectively final。

通常用于封装方法的实现细节。

匿名内部类

匿名内部类是没有名字的类，通常用于简化代码实现。

常见于接口或抽象类的快速实现。

匿名内部类必须继承一个类或实现一个接口。

用于简化代码实现，无需单独定义类。

  • 匿名内部类继承类

Class对象

获取Class对象

  • 通过对象获取
    Class<?> clazz = object.getClass();

  • 通过类获取
    Class<?> clazz = ClassName.class;

  • 通过全类名获取
    Class<?> clazz = Class.forName(String className);

    Class.forName()会在加载类的同时对类进行初始化。

构造方法对象

获取构造方法对象

  • 获取公有构造方法对象的数组
    Constructor<?>[] getConstructors()

  • 获取所有构造方法对象的数组
    Constructor<?>[] getDeclaredConstructors()

  • 获取单个公有构造方法对象
    Constructor<?> getConstructor()

  • 获取单个构造方法对象

    Constructor<?> getDeclaredConstructor()

调用构造方法

newInstance()

  • 调用公有构造方法

  • 调用私有构造方法
    使用setAccessible(true)突破 Java 的访问控制限制，使得私有成员（字段、方法或构造方法）可以被访问。

Field属性对象

获取成员属性对象

  • 获取公有成员属性对象的数组
    Field[] getFields()

  • 获取所有成员属性对象的数组
    Field[] getDeclaredFields()

  • 获取单个公有成员属性对象
    Field[] getField(String name)

  • 获取单个成员属性对象
    Field[] getDeclaredField(String name)

获取和设置属性的值

  • get(Object obj)

  • set(Object obj, Object value)

Method方法对象

获取方法对象

  • 获取公有成员方法对象的数组，包括继承的
    Method[] getMethods()

  • 获取所有成员方法对象的数组，不包括继承的
    Method[] getDeclaredMethods()

  • 获取单个公有成员方法对象
    Method getMethod(String name,Class<?>…parameterTypes)

  • 获取单个成员方法对象
    Method getDeclaredMethod(String name,Class<?>…parameterTypes)

调用方法

Method.invoke()

调用公有方法和私有方法

获取方法参数信息

ClassLoader类加载器对象

获取类加载器对象

getClassLoader()

通过Class对象获取ClassLoader

ClassLoader classLoader = clazz.getClassLoader();

通过类加载器对象加载类

loadClass(String name)

通过ClassLoader对象加载类

loadClass()用于加载类，但不会初始化类。

其他常用方法

  • getName()

    用于获取类、构造方法、方法或字段的名称。

  • getModifiers()

    获取类、方法、字段或构造方法的访问修饰符。

    getModifiers() 返回一个 int 类型的值，这个整数值表示多个修饰符的位运算组合。

  • setAccessible(true)

    用于修改类、方法、字段等成员的访问权限，以便访问被 private、protected 或 default（包内访问权限）修饰的成员。
