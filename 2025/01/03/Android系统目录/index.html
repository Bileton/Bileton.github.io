data

system

app

data/app/存储已安装应用的 APK 文件以及应用的私有数据。

user

  • 在 Android 系统中，/data/user/ 目录是一个用于支持多用户环境的目录。它存储了与每个用户账户相关的数据。

/data/user/ 目录包含每个用户的私有数据，并为每个用户分配一个唯一的 ID（从 0 开始）。

  • 主用户路径/data/user/0
